***Работа с монитором COM порта***
```c++
Serial.begin(9600) // открыть порт (Serial.begin(<скорость>)измеряется в бодах) 

Serial.end() // прекратить связь

Serial.print("Hello") // вывод в порт (только в одну строку)

Serial.println("Hello") // вывод в порт с новой строки 

Serial.println(3.1456, 2) // Serial.println(val, n) вывод val с n количеством знаков после запятой 

Serial.println(34422 , DEC) // вывод с указаным базисом: 
//DEC - десятичный
//HEX - 16-ричная
//OCT - 8-ричная
//BIN - двоичная

Serial.available() // возвращает количество байт, находящихся в буфере приёма и доступных для чтения

Serial.read() // прочитать входящие данные в символьном формате ASCII

```
